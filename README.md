# Fulll - React - Level 2

## Global informations

This is an app that help you to find users on GitHub. You can find any profile by typing nickname or even firstname / lastname.

## Requirement

NodeJS - Yarn or NPM

## Installation

execute it at the root directory

```
$ yarn install
```

## Start the app

```
$ yarn start
```

## Test

```
$ yarn test
```
